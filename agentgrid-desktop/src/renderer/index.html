Install CLIs

Run the command for each CLI you want to use. Requires Node.js 18+ (or Homebrew on macOS).

Codex OpenAI
npm install -g @openai/codex
Claude Code Anthropic
npm install -g @anthropic-ai/claude-code
Gemini Google
npm install -g @google/gemini-cli

After installing, restart AgentGrid or click "Check again" in the System Health section.
