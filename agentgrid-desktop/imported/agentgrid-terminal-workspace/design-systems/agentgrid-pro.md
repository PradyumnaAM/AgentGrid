## Brand & Style

The design system is engineered for technical precision and high-performance workflows. It targets senior developers and engineers who require a tool that feels as robust as a terminal but as refined as a high-end IDE.

The visual direction combines Technical Minimalism with Atmospheric Depth. By utilizing a dark, slate-heavy palette punctuated by intense luminescent accents, the interface evokes the feeling of a high-end command center. Every element is designed for density and utility, ensuring that complex data remains legible and actionable without unnecessary decorative overhead.

Key attributes include Surgical Precision, Operational Power, and Tactile Feedback.

## Typography

Inter is used for primary UI text, navigation, and body copy. JetBrains Mono is used for code blocks, labels, metadata, and tabular data.

## Layout & Spacing

Dense Fluid Grid with a 12-column grid, tight 16px gutters, and a strict 4px baseline grid.

## Components

Primary buttons use Electric Blue fill with white monospaced text. Secondary buttons are ghost-styled with a subtle Slate-700 border that transforms into an Orange Glow on hover. Inputs use a deep Slate-950 background with crisp Electric Blue focus. Tables use horizontal dividers only. Chips use monospaced text inside small rounded containers.
